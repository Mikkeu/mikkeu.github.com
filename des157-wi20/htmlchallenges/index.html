Why we use the Normalize CSS Script

We use the normalize CSS script because it provides a similar "canvas" to work on across different browsers. It is different from doing a CSS reset because it is not leaving a completely blank webpage. Normalize CSS would leave elements that are useful and make creating the webpage easier without having to imput all the fonts or text sizes.

What is Semantic Markup

Semantic markups are tags that show what type of text you are adding like is it a paragraph or the header, etc. It is useful for organization and communicating the information on your webpage. It makes the content on the page easier for search engines to read and make your webpage easier to find online. It will also be easier for you to find errors in your own code.
